# Variables

Declaration: `let name;`

Instantiation: `let name = value;`

Modification: `name = value;`
    
# Operators

## Comparison

Equals: `==`

Greater: `>`

Less: `<`

Greater/equal: `>=`

Less/equal: `<=`

Not Equal: `!=`

## Boolean

And: `and`

Or: `or`

Not: `not`

# Decisions

If block:
```
if comparison1 {
    do something 1;
} elif comarison2 {
    do something 2;
} else {
    do something 3;
}
```

Switch:
```
switch (variable) {
    case comparison1 {do stuff 1}
    case comparison2 {do stuff 2}
    case comparison3 {do stuff 3}
}
```

# Subroutines

Definition:
```
function name(params) {
    some code;
}
```

Calling:
`name(args);`
`variable = name(args);`
